Next recommended hardware test:
Laptop -> FT232RL USB to UART TTL -> STM32 USART1.

UART1 pins:
STM32 PA9  = USART1_TX
STM32 PA10 = USART1_RX
Baudrate   = 115200 8N1

Wiring:
FT232RL TXD -> STM32 PA10 / USART1_RX
FT232RL RXD -> STM32 PA9  / USART1_TX
FT232RL GND -> STM32 GND

Do not connect 5V UART signal to STM32 pins. Use 3.3V TTL.
